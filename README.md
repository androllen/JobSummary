###[内存泄漏-垃圾回收机制](http://www.cnblogs.com/solan/archive/2012/08/24/CSharp11.html)  
系统会给每个app 分配一定的大小  
当app内存超过大小无法释放，导致app崩溃，系统将app进行垃圾回收  
当app内存分配在规定的大小范围内，app在最后exit的时候，系统将app占用的内存进行垃圾回收
